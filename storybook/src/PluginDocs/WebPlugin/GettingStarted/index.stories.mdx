# Getting Started

##

<br/>

This plugin provides a comprehensive **frontend** solution for your application using **Next.js**. It offers three types of templates for you to choose from: **next-gluestack-ts**, **next-tailwind-ts**, and **next-ts**. These templates provide a solid foundation for building your application's **frontend**, with pre-configured settings and libraries to help you get started quickly.
